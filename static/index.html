How to connect the radio
  1. Enable TCP access on the Meshtastic node and put it on the same LAN as this computer.
  2. Create .agent-venv with Python 3.11 or 3.12 and install requirements-agent.txt.
  3. Enter the node hostname or IP above, then run the generated command in a terminal.

127.0.0.1 is correct only when the agent and Moonbird server run on the same computer. Remote operators must use the public Moonbird URL.

macOS/Linux setup: python3.12 -m venv .agent-venv && .agent-venv/bin/pip install -r requirements-agent.txt
